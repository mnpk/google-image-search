# google-image-search

Google Images and Gif Search with Python


## Example
```python
GoogleAPI.search_img("octocat")
```

return:

https://cdn.shopify.com/s/files/1/0051/4802/products/mona-1_large.jpg?v=1511308586


![](https://cdn.shopify.com/s/files/1/0051/4802/products/mona-1_large.jpg?v=1511308586)



## Gif Example
```python
GoogleAPI.search_gif("octocat")
```
